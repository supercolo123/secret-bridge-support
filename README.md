# Secret Bridge – Issue Tracker

This repository is dedicated exclusively to **issue tracking and feature requests** for the IntelliJ plugin:

👉 **Secret Bridge**

## Purpose

This repository is used only for:
- 🐛 Reporting bugs
- 💡 Requesting new features
- ❓ Asking usage-related questions
- 📋 Tracking known issues and their status

## Important

⚠️ This repository does **not contain the source code** of Secret Bridge.  
⚠️ It is not intended for contributions, pull requests, or code submissions.  
⚠️ The plugin’s source code is maintained in a separate private repository.

## Before submitting an issue

Please:
- Check existing issues before creating a new one
- Provide clear steps to reproduce bugs
- Include relevant environment details (IDE version, plugin version, OS)

## Security issues

If you discover a security vulnerability, please do **not** open a public issue.  
Instead, contact the maintainers privately.

## About Secret Bridge

Secret Bridge is an IntelliJ plugin designed to securely manage and bridge secrets within your development workflow.
